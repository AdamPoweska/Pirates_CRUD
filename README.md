Command-line crew management system built in Python, serving as the foundation for a future pirate-themed text adventure game. The application incorporates core CRUD functionality, enabling users to recruit, manage, and remove crew members while tracking associated costs and attributes. It emphasizes object-oriented programming, input validation, menu-driven interactions, and automated testing, showcasing Python fundamentals and software design principles in a practical implementation.
Responsibilities:
• Designed and developed a console-based crew management application demo using Python.  
• Applied Object-Oriented Programming (OOP) principles to create custom classes for modeling crew members and game entities.  
• Implemented full CRUD operations for recruiting, viewing, and removing crew members.  
• Developed robust user input validation for character attributes, skills, professions, and game actions.  
• Created dynamic crew management features, including automatic crew numbering and cost calculations.  
• Designed and implemented profession-based character specialization (e.g., Sailors and Warriors) with unique skill sets.  
• Built a menu-driven navigation system utilizing function dispatching and lambda expressions.  
• Designed formatted terminal output for displaying crew statistics and roster information.  
• Developed game preparation mechanics, including crew budgeting and resource management concepts.  
• Added comprehensive error handling and validation to enhance application stability and user experience.  
• Created automated unit tests using Pytest to ensure the accuracy of validation functions and class behavior.  
• Structured the project to allow future expansion into a complete text-based adventure game.
